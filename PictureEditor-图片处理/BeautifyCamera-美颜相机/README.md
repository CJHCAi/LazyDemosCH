# BeautifyCamera
基于GPUImage实现的美颜相机，常见的功能： 滤镜选择， 聚焦，人脸追踪，闪光灯，延时拍照，前后摄像头切换， 历史浏览，裁剪均有实现
<br>页面布局用的是masonry

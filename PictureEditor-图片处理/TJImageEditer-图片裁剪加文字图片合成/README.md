# TJImageEditer
选取图片进行裁剪，裁剪完毕后可以对图片进行文字和图片编辑，并保存

可以自定义图片编辑区域大小
支持添加文字和图片，对当前编辑图片或文字有旋转缩放拖动的手势支持

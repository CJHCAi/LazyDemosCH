# linjiaMusic
模仿天天动听音乐，数据也是抓的天天动听

主要功能：</br>
1. 音库（新歌首发(专辑)、周排行、歌手、分类、大家在听）</br>
2. 热听（主题歌、轮播图歌单）</br>
3. 搜索（搜索想听的歌曲）</br>
4. 歌单列表</br>
5. 歌曲播放界面（包括整个播放页，以及所有页面下方始终展示播放歌曲）</br>

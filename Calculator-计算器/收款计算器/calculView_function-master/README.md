# calculView_function
iOS 收款计算器界面
一个收款计算器算法,从之前高仿有赞Demo里面抽离的一个界面


* 显示计算记录
* 不能连续输入俩个计算符号
* 小数点的位数不能超过俩位
* 单个小数点的时候不能输入  00
* 点击 + ，直接显示计算结果，正则匹配计算
* 设置最大显示金额
